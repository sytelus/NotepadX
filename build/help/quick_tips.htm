Quick Tips

  • Use following keyboard shortcuts to quickly add notes:
    Insert Key - Add new note
    Delete Key - Delete selected note
    F2 Key - Rename selected note
  • You can change color, font etc of your notes by selecting Note Properties in Edit menu.
  • If you added a textual note and then you wanted it to be say a Worksheet type of note, just go to Edit > Note Properties and select Content Format. Remeber you will loose your existing data!
  • If you added a textual note and then you wanted it to be say a Worksheet type of note, just go to Edit > Note Properties and select Content Format. Remeber you will loose your existing data!
  • Sometime you want to make note readonly so you don't modify it accidently. Use Edit > Make Note Readonly menu for this.
  • Sometime you want to put a password for a note. So if anyone tries to open it, he or she would be asked for a password. Use Edit > Make Note Secure menu for this. Remember, if you forget the password, there's no going back. NotepadX doesn't save your password to get it back. It uses strong encryption that comes with Microsoft .Net Framework so it's very difficult to access the content, if not impossible, unless you have password .
  • NotepadX maintains author information so it is possible to see if anyone else modified anything. Currenly it's all internally stored and may be visible only if you watch Edit > Note Properties XML option.
  • Try Tools > Options menu. You can see the explanation as you highlight various options.
